Improving Content Forms

Hiding Fields

  • hook_form_alter in a custom module
  • Simplify Module

hook_form_alter

Disable a field:


$form['field_myfield']['#disabled'] = TRUE;

Hide a field preserving it's value:


$form['field_myfield']['#access'] = FALSE;
